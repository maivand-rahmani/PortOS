# public

Static assets that Next.js serves from the site root.

## Files

- `file.svg`: generic file icon placeholder.
- `globe.svg`: globe icon from the starter asset set.
- `next.svg`: Next.js logo asset from the starter template.
- `vercel.svg`: Vercel logo asset from the starter template.
- `window.svg`: generic window icon placeholder.

Use this folder for shared static assets that do not belong to a single app module.
