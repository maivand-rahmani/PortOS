# entities

Core domain building blocks for the Portfolio OS.

## Files

- `window.ts`: runtime window data, position, sizing, fullscreen state, and restore metadata.
- `workspace.ts`: virtual desktop, fullscreen-space, and split-view workspace definition types.
- `app.ts`: application metadata, status bar contracts, lazy-loader, and component contracts.
- `process.ts`: process instance and lifecycle types.
- `file-system.ts`: virtual file system node types, content storage, path types, MIME mapping, search result types, and clipboard types.

Entities define the nouns of the system that features and widgets build on top of.
