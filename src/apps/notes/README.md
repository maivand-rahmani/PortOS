# notes

Hand-drawn notebook-style notes app with local persistence, tagging, pinning, and filtered browsing.

## Folders and files

- `index.ts`: app metadata and lazy loader.
- `model/`: note types, normalization, and browser persistence helpers.
- `theme.css`: notebook paper texture and ruled-page visuals for the app shell.
- `ui/`: sketch-style notes workspace with sidebar filters and page editor.
