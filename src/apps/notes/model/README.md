# model

Notes data model and filesystem-backed persistence helpers.

## Files

- `notes-ai-context.ts`: builds the active Notes page context published to the OS AI service.
- `notes-storage.ts`: note types, markdown serialization, filesystem-backed persistence, and directory subscriptions.
- `notes-external-request.ts`: external Notes event contract and request application helpers for create, upsert, and in-place replace flows.
