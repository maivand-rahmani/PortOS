# ui

Calculator UI for the compact workspace and working tape.

## Files

- `calculator-actions.tsx`: copy, Notes, and AI handoff actions for the active calculation.
- `calculator-app.tsx`: focusable calculator workspace with keyboard handling.
- `calculator-display.tsx`: live display and operator feedback.
- `calculator-keypad.tsx`: keypad grid for calculator input.
- `calculator-tape.tsx`: persistent tape list with restore and pin actions.
