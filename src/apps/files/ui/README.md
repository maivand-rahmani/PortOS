# ui

Files app visual components.

## Folders

- `files-app/`: root app shell that composes sidebar, toolbar, breadcrumb, content view, preview, and context menu.
- `files-sidebar/`: sidebar with root directory shortcuts.
- `files-toolbar/`: top toolbar with navigation, create, view mode, and search controls.
- `files-breadcrumb/`: breadcrumb trail for current directory path.
- `files-grid/`: grid view for displaying file system nodes.
- `files-list/`: list/table view for displaying file system nodes.
- `files-preview/`: side panel for file metadata and text content preview.
- `files-context-menu/`: right-click context menu with file operations.
- `files-shared/`: shared helpers, icon components, and formatters used across Files UI.
