# files

File manager app with tree navigation, grid and list views, search, clipboard operations, and inline text preview.

## Folders and files

- `index.ts`: app metadata, status bar config, and lazy loader.
- `icon.tsx`: Liquid Glass SVG folder icon.
- `theme.css`: CSS custom properties for the Files color scheme and layout tokens.
- `model/`: state management hook for navigation, selection, CRUD, and clipboard.
- `ui/`: visual components split by responsibility (sidebar, toolbar, breadcrumb, grid, list, preview, context menu).
