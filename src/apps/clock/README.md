# clock

World clock app with searchable worldwide timezone data, favorites, layered glass panels, and live timezone tracking.

## Folders and files

- `index.ts`: app metadata, larger default window sizing, and fullscreen-on-launch behavior.
- `theme.css`: cinematic dark glass theme and dial depth styling.
- `model/`: timezone data helpers built from the installed timezone library.
- `ui/`: full app workspace with search directory, favorites, hero clock, and tracked city wall.
