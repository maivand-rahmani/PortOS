# Editor UI

Visual components for the Editor app.

## Components

| Folder | Purpose |
|--------|---------|
| `editor-app/` | Root component that composes all sub-components into the editor layout |
| `editor-toolbar/` | Toolbar with save/undo/redo, mode-specific formatting buttons, preview toggle |
| `editor-canvas/` | Textarea editing surface with line numbers and scroll sync |
| `editor-status/` | Status bar with file path, mode, word/line/char counts, save status, auto-save toggle |
| `editor-preview/` | Markdown live preview with custom lightweight renderer |
