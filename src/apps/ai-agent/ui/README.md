# ui

Visual surface for the AI agent app.

## Files

- `AgentHandoffBar.tsx`: live handoff banner for externally triggered agent requests.
- `AgentLaunchpad.tsx`: fresh-session workflow cards and quick-start prompts.
- `ChatWindow.tsx`: main app layout and orchestration.
- `MessageBubble.tsx`: animated bubble styles for user, assistant, and system notes.
- `Input.tsx`: sketch-style composer with send action and command chips.

This folder keeps the interface split into focused pieces so the app can scale without turning into one large chat component.
