# ui

Blog reader UI.

## Files

- `blog-app.tsx`: orchestrates blog loading, local reading state, and cross-app handoff actions.
- `blog-post-list.tsx`: searchable post index with queue and completion badges.
- `blog-post-view.tsx`: active post reader with highlight capture and OS handoff actions.
- `blog-reader-header.tsx`: reader summary, search input, and session status messaging.
- `blog-reading-sidebar.tsx`: queue and per-post highlight panels for the current reading workflow.
