# ui

System Info UI.

## Files

- `system-info-app.tsx`: top-level app composition for all System Info sections.
- `system-info-diagnostics.tsx`: incident rail with runtime health, diagnostics, and export actions.
- `system-info-overview.tsx`: masthead, meters, and charts.
- `system-info-primitives.tsx`: shared section headers, detail rows, tooltips, and action buttons for this app.
- `system-info-processes.tsx`: process list and selected-target operator controls.
- `system-info-windows.tsx`: window ledger chart and rows.
