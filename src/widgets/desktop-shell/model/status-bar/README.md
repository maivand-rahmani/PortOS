# status-bar

Derived menu bar models and safe command execution helpers for the desktop shell.

## Files

- `status-bar.model.ts`: builds shell menu bar data from active runtime state and app metadata.
- `status-bar.commands.ts`: runs declarative status bar commands without storing callbacks in app configs.
- `status-bar.types.ts`: shared status bar model and runner types.
- `index.ts`: public exports for the status bar model layer.
