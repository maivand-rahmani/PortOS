# window-manager-surface

Composition root for the workspace track, window surfaces, split-view UI, snap guides,
file-drop overlays, and dock context menus.

## Files

- `window-manager-surface.tsx`: renders the sliding workspace track with `<AnimatePresence>`,
  `<WindowSurface>` instances, `<SplitViewDivider>`, `<SplitViewPicker>`,
  `<SnapGuideOverlay>`, `<FileDropOverlay>`, and `<DockMenu>`.
