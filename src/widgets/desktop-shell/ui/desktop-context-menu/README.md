# desktop-context-menu

Desktop right-click context menu for the desktop shell.

## Files

- `desktop-context-menu.tsx`: root AnimatePresence wrapper with glass-morphism styling, viewport clamping, and item rendering.
- `desktop-context-menu-item.tsx`: single menu row for action and submenu items with hover, destructive, disabled, and checked states.
- `desktop-context-menu-separator.tsx`: thin line divider between menu item groups.
- `index.ts`: barrel re-export.
