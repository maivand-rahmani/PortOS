# app-data

Static application content — blog posts, docs helpers, project profile, and wallpapers.

## Files

- `app-logic.ts`: blog post data, contact validation, terminal command runner, and file system helpers.
- `docs.ts`: shared helpers for docs navigation and heading IDs.
- `project-data.ts`: typed access to profile basics (name, title, location, contact links).
- `wallpapers.ts`: wallpaper definitions used by the settings and desktop wallpaper system.
- `index.ts`: re-exports the public surface of all app-data utilities.
