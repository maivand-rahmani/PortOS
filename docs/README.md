# docs

Canonical project documentation for Portfolio OS.

## Folders

- `roadmap/`: implementation stages, priorities, and completion tracking.
- `style/`: UI, UX, and visual-system rules.
- `maivand/`: all about me and my personal projects, including the ones related to Portfolio OS.
- `project/`: all about Portfolio OS, including design documents, research, and development notes.

The tracked documentation path in this repository is lowercase `docs/`. Every documentation folder must include a `README.md` that explains what it contains.
